# Bulk Vegetable & Fruit Ordering Platform (Frontend)

## Tech Stack
- React.js
- React Router DOM
- HTML
- CSS
- JavaScript

##  Features Implemented
### Buyer
- View vegetable and fruit catalogue
- Place bulk order using order form
- Track order status (UI flow)

### Admin
- View orders page
- Update order status (planned)

## ⚙️ Backend Implementation Status
As per the assignment requirements, backend APIs using Node.js, Express, and SQLite are planned.
For this submission, frontend is implemented using mock/static data to demonstrate the complete UI flow.
